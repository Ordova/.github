# Ordova

Ordova is in early development. Here is the idea behind it.

Most AI tools hand you someone else's defaults, inside someone else's walls. Ordova is being built so you author your own: the principles your AI works by, set and enforced by you, not by the vendor whose tool you happen to be using. You write the rules.

The name is claimed and the work is underway. There is nothing to install yet.

An Async Digital Ltd project. [ordova.ai](https://ordova.ai)
